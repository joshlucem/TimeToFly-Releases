### Requirements
- **Minecraft:** 1.18 - 1.21
- **Java:** 17 or higher
- **Server:** Spigot, Paper or forks

### Installation

1. Download `TimeToFly-1.1.1.jar` from the releases folder
2. Place it in your `plugins/` folder
3. Restart your server
4. Configure in `plugins/TimeToFly/`

### Configuration for MySQL (Optional)

```yaml
database:
  type: mysql
  mysql:
    host: localhost
    port: 3306
    database: timetofly
    username: root
    password: "your_password"
```
